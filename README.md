# Lab 1: Java Data Processing

A CS2 Java lab that practices reading input data and applying foundational data-processing logic.

## Highlights
Uses an input.txt file.
Practices Java parsing and output.
Prepared as a standalone lab folder.
Introduces the CS2 lab workflow.

## Overview
Lab 1 starts the CS2 sequence with a focused Java data-processing assignment. The program reads from a local input file and applies the lab's logic to produce console output.
The project reinforces careful file placement, compilation, and program execution from the correct working directory.

## Why This Project
This project sets up the workflow used by later data structures labs: keep input files with the source, compile locally, and reason through program output from structured data.

## Usage
Compile: Compile the Java source with javac *.java.
Run: Execute the program with java Lab1_Cardenas.
Keep input.txt in the same folder before running.

## Installation
Ensure you have the Java Development Kit (JDK) installed on your system.
Open a terminal in the lab folder before compiling so the program can find any required local text files.

## Architecture
File Structure: The project contains Lab1_Cardenas.java and input.txt.
Components: Lab1_Cardenas.java is the driver. input.txt provides the data used by the lab.

## Future Improvements
